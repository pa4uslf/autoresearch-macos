# 债务处理副业推广知识库导航

## 1. 这组文档怎么分工

这一组文档已经收敛成“总览 + 模板 + 执行”三层结构：

1. `0` 负责总览和边界。
2. `1-3` 负责内容与沟通模板。
3. `4-5` 负责执行和场景分诊。

如果以后继续扩展，也尽量沿着这个结构来，不要再把原则、模板、执行细节混写到同一篇里。
