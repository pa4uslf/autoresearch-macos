# 债务处理副业平台推广实操手册

> 适用背景：本人以前端获客身份，为湖南鹏橙法律咨询有限公司合作开展债务处理相关业务；前端承接方式为个人账号内容获客 + 个人微信沟通 + 后端公司/合作律所签约；本人仅持有公司内部推广授权，不持有国家或第三方平台认可的法律执业/平台行业资质。

3. 绝对不能越过的边界。
4. 配套模板库入口。

具体执行内容不再写在这里，统一放到 `7-11` 文档中。

## 3. 当前模式的核心约束

当前模式不是“律所官方直营获客”，而是：

1. 前台接触用户的人通常是我个人。
2. 合作主体是湖南鹏橙法律咨询有限公司，不是律师事务所。
3. 公司背后虽有合作律所，但用户第一触点不是律所官方入口。
4. 我本人只有公司内部推广授权，不具备平台可直接认可的行业身份。
5. 当前结算方式是签约后提成，短期内不改变。

平台真正会看的，不是内部合作关系，而是：

1. 账号主体是谁。
2. 内容是谁发的。
3. 联系方式是谁的。
4. 是否存在站外导流到个人微信。
5. 是否存在主体混淆、夸大承诺、结果承诺、敏感词触发。

一句话结论：

> 只要继续使用“个人号 + 个人微信 + 债务处理类转化”，小红书和抖音就天然属于高风控平台，不能把它们当作最稳定的主阵地。

### 第一梯队：优先投入

2. 知乎
3. 公众号 / 视频号

### 第二梯队：谨慎测试

1. 小红书
2. 抖音

### 第三梯队：暂不作为重点

1. 闲鱼

## 5. 平台分工

不同平台只做不同的事，不要混着用：

1. 百度 SEO / 落地页：接搜索需求，承接高意向用户。
2. 知乎：做问题澄清、长尾流量、专业感建立。
3. 公众号 / 视频号：做资料沉淀和微信承接。
4. 小红书：做低刺激曝光和避坑提醒。
5. 抖音：做问答型短视频测试。

## 6. 永远不要越过的边界

无论在哪个平台，以下边界必须长期坚持：

1. 不冒充律师，不暗示自己是执业律师。
2. 不冒充律所官方，不暗示自己是律所接案口。
3. 不把法律咨询公司写成律师事务所。
4. 不做结果承诺。
5. 不公开大面积留微信号、二维码、手机号。
6. 不在公开内容里高频刺激“赶紧加我”“私我包解决”。
7. 不把公司内部授权说成国家资质、官方资质、平台认证。

## 7. 内容总原则

公开内容统一遵守一句话：

> 不卖“处理结果”，只卖“认知澄清 + 阶段判断 + 风险边界”。

最适合的内容方向只有四类：

1. 问题澄清型
2. 流程拆解型
3. 避坑提醒型
4. 情绪缓冲型

## 8. 绝对不要碰的高风险表达

以下表达统一视为禁用：

1. 包停催
2. 包减免
3. 包延期
4. 包协商成功
5. 包上岸
6. 100% 成功
7. 征信修复
8. 洗白征信
9. 内部渠道
10. 银行关系
11. 法院有人
12. 律所内部资源
13. 一分钱不用还
14. 全额减免
15. 停息挂账包办

## 9. 最精简执行顺序

如果后面只记一个顺序，就记这个：

1. 先看用户来自哪个平台。
2. 先判断用户属于什么场景。
3. 先把问题分类，再决定怎么回。
4. 先做身份披露，再做信息采集。
5. 先判断，后引导，最后才谈是否继续推进。

## 10. 推荐阅读顺序

第一次用时，按这个顺序读：

1. 先看这篇 `6`，明确边界和平台顺序。
2. 再看 `11`，学会识别咨询场景。
3. 再看 `9`，学会怎么私信和微信开口。
4. 再看 `7` 和 `8`，开始写内容。
5. 最后看 `10`，把节奏固定下来。
